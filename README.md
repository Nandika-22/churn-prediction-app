Customer Churn Prediction App

A simple Machine Learning + Deep Learning web app that predicts if a customer will churn (leave) or stay, using their account and usage details.

Built to learn the complete AI project lifecycle — from data preprocessing to deployment.

Live Demo & Code:
Streamlit App: [https://churn-prediction-app-ep2xnsp338s4ropnzth7uv.streamlit.app/]

Why This Matters:

Customer churn hurts subscription-based businesses.
This app predicts churn before it happens, helping companies take action to retain customers.

Tech Stack:

Python | Pandas, NumPy

Scikit-learn | TensorFlow / Keras

Streamlit | Joblib | Google Colab

How It Works:

Enter customer details (tenure, services, charges…)

Data is preprocessed using the trained scaler

Neural network predicts churn

Results show:

Customer is likely to churn

Customer will stay

Features:

Clean and simple UI

Real-time prediction

End-to-end ML pipeline

Beginner-friendly deployment

Model Overview:

Type: Neural Network (Deep Learning)

Problem: Binary Classification

Target: Churn (Yes/No)

Metrics: Accuracy, Loss
